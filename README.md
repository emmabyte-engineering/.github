# Emmabyte Engineering

## About Us

Emmabyte Engineering is a software engineering organization focused on building practical tools and applications across the stack. We work with modern technologies including **Go**, **TypeScript**, **Svelte**, and **Vue**, and we're passionate about developer experience, infrastructure, and clean software design.

Our public repositories represent a mix of open-source tools, experimental projects, and reference implementations. We also maintain private repositories for client and internal work.
